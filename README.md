# Big Data Analysis Using PySpark

## Description
This project demonstrates big data analytics on a large dataset using PySpark.
It performs filtering, aggregation, sorting, and scalability analysis using
distributed processing.

## Files
- sales_data.csv → Input dataset
- big_data_analysis.py → PySpark analytics code
- requirements.txt → Dependencies

## How to Run
1. Install Java (JDK 8 or 11)
2. Install Python 3.x
3. Install PySpark: pip install pyspark
4. Run the program:
   python big_data_analysis.py

## Output
- Region-wise total sales
- Product-wise average sales
- High sales count
- Execution time
